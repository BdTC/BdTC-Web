Artistas, diseñadores, desarrolladores, comunicadores, músicos, arquitectos, emprendedores ...

¿Cómo funciona?

Los usuarios registrados en la plataforma pueden intercambiar sus habilidades, utilizando el tiempo como unidad de medida. Desarrollamos un sistema de naipes especiales que facilitan compartir el talento profesional, mediante el intercambio de servicios y el crédito mutuo.

La unidad de medida es el tiempo que lleva realizar una tarea y el propósito, impulsar proyectos sin dinero.

Naipes especiales con habilidades estandarizadas

Cada naipe posee una habilidad específica y un valor expresado en tiempo.
Su objetivo es visibilizar como se pueden gastar las horas y agilizar los intercambios.

“Puedo impulsar mi proyecto,
consiguiendo lo que necesito con tiempo”.
